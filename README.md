## Development

* `git clone https://github.com/cball/justa-table.git`
* `cd justa-table`
* `npm install`
* `bower install`

## Previewing

* `ember server`
* Visit your app at [http://localhost:4200](http://localhost:4200).

## Running Tests

* `ember test`
* `ember test --server`

## Building

* `ember build`
